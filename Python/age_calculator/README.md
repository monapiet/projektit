# Age Calculator

## Description / Kuvaus

### English:
Age Calculator is a Python program that calculates a user's age in years, months, and days. The user inputs their birth date, and the program computes their age based on the current date.

### Suomi:  
Age Calculator on Python-ohjelma, joka laskee käyttäjän iän vuosina, kuukausina ja päivinä. Käyttäjä syöttää syntymäpäivänsä, ja ohjelma laskee iän kuluvan päivän perusteella.

---

## Installation & Usage / Asennus ja käyttö

### English:  
1. Clone or download the project.  
2. Run the program using Python:
 
  python age_calculator.py
  Enter your birth year, month, and day when prompted.

  The program will display your age in years, months, and days.

### Suomi:

1.Lataa tai kloonaa projekti.
2 Suorita ohjelma Pythonilla:

  python age_calculator.py
  Syötä syntymävuosi, -kuukausi ja -päivä, kun ohjelma pyytää.

  Ohjelma tulostaa iän vuosina, kuukausina ja päivinä.
