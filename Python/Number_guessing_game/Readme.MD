##  Number Guessing Game
# Description / Kuvaus
English:
Number Guessing Game is a Python program where the player tries to guess a randomly generated number between 1 and 100.
The program gives colored feedback using the Colorama library to indicate if the guess is too high, too low, or correct.
The game also counts the number of attempts until the correct number is guessed.
To play, install Python and the Colorama library if needed, then run the program and follow the prompts.

Suomi:
Number Guessing Game on Python-ohjelma, jossa pelaajan tehtävänä on arvata satunnaisesti arvottu luku väliltä 1–100.
Ohjelma antaa värillistä palautetta Colorama-kirjaston avulla: liian suuri, liian pieni tai oikein.
Ohjelma myös laskee, kuinka monta yritystä pelaaja tarvitsee oikean luvun arvaukseen.
Pelaamiseen tarvitset Pythonin ja tarvittaessa Colorama-kirjaston. Suorita ohjelma ja seuraa näytön ohjeita.

Installation & Usage / Asennus ja käyttö
English:

Download or clone the project to your computer.
Make sure Python is installed.
If Colorama is not installed, install it using pip install colorama.
Run the program by executing python guessing_game.py in your command line.
Enter numbers between 1 and 100 when prompted. The program will give feedback and count your attempts until you guess the correct number.
Suomi:

Lataa tai kloonaa projekti tietokoneellesi.
Varmista, että Python on asennettu.
Jos Colorama ei ole asennettuna, asenna se komennolla pip install colorama.
Suorita ohjelma komennolla python guessing_game.py.
Syötä lukuja väliltä 1–100, kun ohjelma pyytää. Ohjelma antaa palautetta ja laskee yritysten määrän, kunnes arvaat oikein.
